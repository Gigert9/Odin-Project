To create the link below we used a standard href attribute within an anchor tag. But we also added the rel attribute. By adding rel="noopener noreferrer" to the anchor tag we accomplish two things:

  • noopener: Ensures that a link opened in a new tab or window cannot interact with or access the original page.
  • noreferrer: Provides both privacy and security by preventing the new page from knowing where the user came from (hiding the referrer) and also includes the behavior of noopener, preventing the new page from accessing the original page.

Finally, we also include the target attribute in order to open the new page in a separate tab. The full attribute is target="_blank".
